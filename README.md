# Elements of Topology, Differential Geometry and General Relativity for Physicists: A Mathematica-based Tutorial Approach

*Supplementary Mathematica notebooks for the book of the same title.*

This repository collects the **Wolfram Mathematica** notebooks that accompany the book
*Elements of Topology, Differential Geometry and General Relativity for Physicists: A Mathematica‑based
Tutorial Approach*. They illustrate, compute, and visualize the ideas developed in the text — from open
covers, homotopy, manifolds and exterior calculus to curvature tensors, geodesics, Lie groups, the Hopf
fibration and Berry curvature. Each notebook is self‑contained and is meant to be read and run
cell‑by‑cell alongside the corresponding material in the book.

> Developed and tested with **Mathematica 12.3**. All required add‑on packages are bundled in
> [`Packages/`](Packages) — no separate installation is needed.

## Requirements

- **Wolfram Mathematica 12.3** (recommended). Most notebooks also run in later versions, but the bundled
  packages were checked against 12.3.
- A working LaTeX + Ghostscript installation is required **only** for `Visualize_Berry_curvature_z.nb`,
  which uses the [MaTeX](https://github.com/szhorvat/MaTeX) package for typeset plot labels.

## Getting started

1. Clone the repository (keeping the `Packages/` folder next to the notebooks):
   ```bash
   git clone git@github.com:diffgeo-topology-mathematica/DiffGeo_Topology_in_Mathematica.git
   ```
2. Open any notebook in Mathematica and evaluate it top‑to‑bottom (**Evaluation ▸ Evaluate Notebook**).

Notebooks that need an add‑on package load it from the local `Packages/` folder using a relative path,
so **run them from their location inside the repository** (the loader relies on `NotebookDirectory[]`).
The typical first cells look like:

```mathematica
SetDirectory[NotebookDirectory[] <> "/Packages/"];
<< DifferentialForms.m           (* or  << Ricci.m ,  << Difform.m  *)
```

and for the tensor‑calculus notebooks:

```mathematica
SetDirectory[NotebookDirectory[] <> "/Packages/RGTC/"];
Get["EDCRGTCcode.m"];
```

## Notes

- A few notebooks are large (multiple MB) because they store rendered 3‑D graphics — e.g.
  `hopf_fibration_visualization.nb`, `berry_curvature.nb`, `Visualize_Berry_curvature_z.nb`,
  `Manifold_examples.nb`, `Vector_fields.nb`, and `SphericalCoordinates.nb`. Re‑evaluating them
  regenerates the figures.
- `Packages/DifferentialForms_old.m` is a previous version kept for reference; the active package is
  `Packages/DifferentialForms.m`.
